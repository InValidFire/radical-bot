# Radical Bot

## Features
- Manage the Minecraft server process
    - **Start/stop the server**
    - **Update the server**
    - **Restart the server**
    - **Run commands on the server**
- Manage Minecraft server backups
    - **Create backups**
    - **Restore backups**
    - **Automatically create backups on a timer**
    - **Listing backups**
    - **Delete backups**
    - **Upload backups to DigitalOcean**
